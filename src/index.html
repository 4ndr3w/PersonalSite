In-development. Controlled by ROS on a Pi and uses a RPLIDAR to build a map of it's environment. The end goal for this project is to use SLAM and path planning to drive between waypoints while avoiding obstructions.

Ball Shooting Robot

In-development. Currently, is able to shoot a 4in poof ball into a Nerf basketball net autonomously from any reasonable distance using computer vision. I am currently working on using computer vision to locate balls and autonomously acquire them. The end goal for this project is be able to autonomously acquire and shoot balls in a small court.

OmniBot

Controlled by custom software written in C++ running on a Pi. The robot uses odometry and an IMU to maintain a (x,y) position estimate relative to it's starting location. Given a destination (x,y), it can drive to that location. This platform has also been used for some simple computer vision demos with the Pixy camera.
